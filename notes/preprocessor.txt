The preprocessor examines the code before actual compilation of code begins and resolves all 
these directives before any code is actually generated by regular statements.

Preprocessor directives are the lines in the code that are preceded by (#)
are in one line only,no semicolon needed

macros definitions
#define ARRAY_SIZE 10
int arr1[ARRAY_SIZE];
int arr2[ARRAY_SIZE];

it is equivalent to 
arr1[10],arr2[10]

similarly we can define functions also(small)

#define getmax(a,b) a>b?a:b

it does not affect the block structure,they remain until it is undefined with - 
#undef ARRAY_SIZE

CONDITIONAL DIRECTIVES - 
#ifdef - compiles only if the macro specified as parameter is defined
#ifndef - compiles only if the macro specified as parameter is not defined
#endif
#if 
#else
#elif

#ifdef -> #if defined or #ifndef -> #if !defined
example- 
#if TABLE_SIZE>200
#undef TABLE_SIZE
#define TABLE_SIZE 200
 
#elif TABLE_SIZE<50
#undef TABLE_SIZE
#define TABLE_SIZE 50
 
#else
#undef TABLE_SIZE
#define TABLE_SIZE 100
#endif
 
int table[TABLE_SIZE]
